# Dicekey Games Website - About Page Added

수정 사항:

1. 상단 `ABOUT` 클릭 시 `about.html` 회사소개 페이지로 이동합니다.
2. 회사소개 페이지는 기존 Dicekey Games 사이트 분위기에 맞춰 다크 판타지 / 골드 톤으로 제작했습니다.
3. 내용은 기존 비즈니스 플랜을 바탕으로 구성했습니다.
4. 첫 화면과 스크롤 화면 이미지는 그대로 유지됩니다.

업로드 방법:

- 이 폴더 안의 `index.html`, `about.html`, `style.css`, `assets` 폴더를 기존 GitHub 저장소에 덮어쓰기 업로드합니다.
- GitHub에서 같은 이름 파일은 Replace / 덮어쓰기 하면 됩니다.
- 업로드 후 `Commit changes`를 누릅니다.
- 사이트 주소에서 새로고침합니다.
